Evento Descripción Ejecución
Onchange El evento onchange se produce cuando el valor de un elemento se ha cambiado. Sirve para botones de radio y casillas de verificación.
Onclick El evento onclick se produce cuando el usuario ace click sobre un elemento

Onfocus El evento onfocus se produce cuando un elemento recibe el foco.
Onsubmit El evento onsubmit se produce cuando se envía un formulario
Onunload El evento onsubmit se produce una vez que la pagina ha descargado, también cuando el usuario se desplaza fuera de la página,depende de la configuración del navegador para que funcione
Ondbclick El evento ondbclick se produce cuando el usuario hace doble click sobre un elemento
Onreset El evento onreset se produce cuando una forma es puesta a cero.
Onkeyup El evento onkeyup se produce cuando el usuario suelta una tecla
Onselect El evento onselect se produce después de algún texto ha sido seleccionado en un elemento.
Onkeypress El evento onkeypress se produce cuando el usuario pulsa una tecla (No dispara todas la claves, ALT, CTRL,SHIFT,CES), el orden entre eventos es; onkeydown,onkeypress,onkeyup
